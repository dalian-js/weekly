- ⚽ 实时协作： 我们可以一起实时创建内容。
- 🛠 自定义插件： 构建自定义插件来完成你的任务。
- 🤖 自动化支持： 连接一切组件，并在 5 秒内实现自动化。
- 🖥 本地部署： 支持Docker和k8s
- 📝 Page支持： 创建内容丰富且 UI 友好的工具的基础。
- 🎨 由 ILLA Design 提供支持： 组件不应限制您的想象力。
