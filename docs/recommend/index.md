![tReader](https://images-1252557999.file.myqcloud.com/uPic/PMMlRM.jpg)
基本功能如下：

- 手机端适配，阅读效果优秀
- 点左上角的添加按钮可以添加文件
- 文件可以从本机、iCloud 或者 OneDrive、MEGA 之类的地方选择
- 如果文件打开乱码，请考虑使用 UTF-8 保存文件
- 文件列表，左滑、右键或按Delete可以显示删除按钮
- 书签
- 搜索
- 目录等
