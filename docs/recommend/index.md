### [M3U8-Downloader](https://github.com/HeiSir2014/M3U8-Downloader)

| 功能                 | 支持       |
| -------------------- | ---------- |
| HLS协议点播源        | ✓          |
| 自定义Http协议头下载 | ✓          |
| 自定义KEY和IV解密    | ✓          |
| 本地M3U8文件下载     | ✓          |
| M3U8 直播源          | ✓          |
| 标准 AES-128-CBC加密 | ✓          |
| 标准 AES-196-CBC加密 | ✓          |
| 标准 AES-256-CBC加密 | ✓          |
| 非标准 AES-*-CBC加密 | ㄨ(可定制) |
| 网页嗅探视频源       | ✓          |
