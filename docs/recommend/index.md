| -------------------- | ---------- |

| 本地M3U8文件下载     | ✓          |
| M3U8 直播源          | ✓          |
| 标准 AES-128-CBC加密 | ✓          |
| 标准 AES-196-CBC加密 | ✓          |
| 标准 AES-256-CBC加密 | ✓          |
| 非标准 AES-*-CBC加密 | ㄨ(可定制) |
| 网页嗅探视频源       | ✓          |
